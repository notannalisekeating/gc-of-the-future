# GC of The Future - Contract Dashboard

A modern, Neo-Brutalist styled contract management dashboard built with React and Tailwind CSS. This single-file HTML application provides comprehensive insights into contract health, exposure analysis, and organizational metrics.

## Features

- **Contract Queue**: High-density contract list with status tracking, SLA monitoring, and risk assessment
- **Exposure Analysis**: Interactive heatmap showing risk exposure across departments and contract clauses
- **Org Health Metrics**: Key performance indicators including average time to contract, total ARR, and contract review statistics
- **Risk Analysis**: Detailed risk assessment with predictive intelligence for forecasting and capacity planning
- **Dark Mode**: Full dark mode support with improved contrast and usability
- **Mobile Responsive**: Fully responsive design that works on all device sizes
- **Interactive Modals**: Detailed views for exposure analysis, risk assessment, and contract details

## Quick Start

Simply open `dashboard.html` in your web browser. No build process or dependencies required!

The dashboard uses:
- React (via CDN)
- Tailwind CSS (via CDN)
- Google Fonts (Archivo Black, Inter)

## Usage

1. Open `dashboard.html` in any modern web browser
2. Click on heatmap cells to view detailed exposure analysis
3. Click on risk scores to see detailed risk assessment and predictive intelligence
4. Click on contract rows to view full contract details in the side drawer
5. Use the dark mode toggle in the top navigation bar

## Key Components

- **KPI Grid**: 4-column grid showing active contracts, SLA risks, pending markups, and average deviation
- **Exposure Heatmap**: Interactive table showing risk exposure by department and clause type
- **Org Metrics**: Comprehensive organizational health metrics
- **Contract Table**: Detailed contract list with actions (Deal Memo, Send, View, Edit)
- **Detail Drawer**: Side panel showing contract timeline, integrations, playbooks, and notes

## Design System

This dashboard implements a Neo-Brutalist design system with:
- Bold, high-saturation color palette
- Hard shadows (4px offset, no blur)
- Thick black borders (2-3px)
- Uppercase headings with condensed sans-serif fonts
- Playful geometry and rounded corners
